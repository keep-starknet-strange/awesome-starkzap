# 🎮 Koool_Shit - On-Chain Gaming

A blockchain-enabled game with permanent on-chain score storage powered by Starknet.

## ✨ Features

- 🎯 **On-Chain High Scores**: Permanent, verifiable leaderboards stored on Starknet blockchain
- 🎮 **Gasless Gaming**: Players pay ZERO gas fees thanks to Cartridge Controller
- 🔐 **Social Login**: Connect with Google, Twitter, or passkey - no crypto knowledge needed
- 🏆 **Global Leaderboard**: Compete with players worldwide for the top spot
- 📊 **Player Stats**: Track personal bests, levels, deaths, and coins
- ⚡ **Real-Time Updates**: Instant score verification and leaderboard updates

## 🚀 Quick Start

### For Players

1. Visit the game
2. Click "Connect Wallet"
3. Sign in with Google/Twitter/passkey
4. Play and watch your scores get saved on-chain forever!

## 📦 What's Inside

- **Cairo Smart Contract**: GameScores contract deployed on Starknet
- **React Frontend**: Modern UI with wallet integration
- **StarkZap SDK**: Seamless blockchain interactions
- **Cartridge Controller**: Gasless transaction support

## 🛠️ Tech Stack

- **Blockchain**: Starknet (Layer 2 on Ethereum)
- **Smart Contracts**: Cairo 2.16.1
- **Frontend**: React 19 + TypeScript
- **Wallet**: Cartridge Controller (gasless)
- **SDK**: StarkZap

## 🎯 How It Works

1. **Connect**: Players connect using familiar social logins
2. **Play**: Complete levels, avoid deaths, collect coins
3. **Submit**: Scores automatically saved on-chain (gasless)
4. **Compete**: Global leaderboard shows world records
5. **Verify**: All scores verifiable on Starknet explorer

## 🌟 Why Blockchain?

- ✅ **Permanent**: Scores can never be deleted or lost
- ✅ **Verifiable**: Anyone can verify legitimate scores
- ✅ **Tamper-Proof**: Impossible to fake or modify scores
- ✅ **Portable**: Players own their gaming history
- ✅ **Future-Ready**: Enable NFT rewards, tokens, achievements
